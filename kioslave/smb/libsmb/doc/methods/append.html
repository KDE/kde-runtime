char *append(const char* URL, const char *string)

Method Description

Use this function to append a string to a URL. '.' and '..' will be interpreted, and the URL returned will always be a valid URL.

URL is the base URL.

string is the string to append to the base URL. It should be a valid URL component (in fact, the resulting URL should be valid), or 0 will be returned. This can be useful precisely if you want to test the validity of the string !

A valid URL will be returned allocated with new, or 0 on error.
