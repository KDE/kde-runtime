SMBShareList *getShareList(const char *hostname, const char *user=0, const char *password=0)

Method Description

This function get the share list of a SMB server. For servers that manage shares with user accounts, the user will need to log in.
Note that as usual, if a user/password is required and not specified, the callback will be called to get them. The problem is that some servers just return a null list when an anonymous access is done, and there is no way of knowing if a user/password should be asked then or if there is really no share available.

hostname is the NetBIOS host name, coded in ASCIIZ, of the SMB server to list.

user is the user name that will be used for login.

password is the user's password that will be used for login.
