SMBIO(char* (*getFunc)(const char*) = 0, const char *hostname=0)

Method Description

The constructor can take up to 2 arguments.

GetFunc is used by the library for retrieving the user name and passwords.
It should basically print the message it is given as an argument, and return a char* answer allocated with new.
This answer will be deallocated by the library when it's not needed anymore (just being paranoïde about passwords...)

hostname is our NetBIOS host name, coded in ASCIIZ. The library will connect to shares using this name if specified, regardless of what name Samba could have already set to the machine. This might induce problems (didn't happen for me however), but one can find it really fun to connect under a dummy name !
