int login(const char* user=0, const char* password=0)

Method Description

login is used to specify a user/password for the current host. It is called internally by the library.
You don't have to use this function for usual operations on files, but only if you wish to manage the sessions by yourself.
If you do so nonetheless, this function should be called immediately after openSession.
If iether user or password isn't specified and needed, it will be retrieved through the callback.

This function detects if the server require encryption, and use it if this is the case.

user is the name the user want to send to the server. This should be an ASCIIZ string.

password is the password for the user account. This should be an ASCIIZ string.

This function returns 0 on success, -1 on error.
