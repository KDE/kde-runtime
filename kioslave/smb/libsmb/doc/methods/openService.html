int openService(const char* service, const char* password=0, uint8 type=SMB_DISK)

Method Description

Use this function to open a service (=share) on a SMB server. This function should only be called when a session has been successfully set up.
You don't have to use this function for usual operations on files, but only if you wish to manage the sessions by yourself.

service is the name of the service/share to open, coded in ASCIIZ. It is relative to the current SMB server.

password is the password required to access this share, if one is needed. It should be in plain ASCIIZ text, and will be encrypted if the server requires so.

type is the type of the ressource. Valid types are :
    SMB_DISK : The usual kind of share.
    SMB_PRINT_QUEUE : A print queue is a kind of file that can be opened for writing, and everything written in it is sent to a printer.
    SMB_DEVICE : Not very clear.
    SMB_IPC : Communication between processes. Used for example for getting the share list.
At present, only SMB_DISK and SMB_IPC have been implemented and tested.

This function returns 0 on success, -1 on error.
