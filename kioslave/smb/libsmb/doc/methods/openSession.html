int openSession(const char *hostname)

Method Description

openSession opens a new session with hostname. It closes the previous opened session if any.
Nothing is specified concerning login yet, for this you need to use login after calling this function.
You don't have to use this function for usual operations on files, but only if you wish to manage the sessions by yourself.

hosname is the NetBIOS name of the host with which the new session is opened.

This function returns 0 on success, -1 on error.
