void setOurName(const char *name)

Method Description

Use this to specify our NetBIOS name.
The library will connect to shares using this name if specified, regardless of what name Samba could have already set to the machine. This might induce problems (didn't happen for me however), but one can find it really fun to connect under a dummy name !

name is our NetBIOS host name, coded in ASCIIZ.
