int rmdir(const char *pathname)

Method Description

This function will remove the directory pathname if it exists.
Notes :
    . The directory must be empty for this function to succeed, and of course it's not possible to remove a workgroup, host or share !
    . The SMB server must have permission to delete the directory.



pathname is the URL of the directory to destroy.

0 is returned on success, -1 on error.
