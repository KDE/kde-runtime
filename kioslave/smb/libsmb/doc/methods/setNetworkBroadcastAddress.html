int setNetworkBroadcastAddress(uint32 addr)
int setNetworkBroadcastAddress(const char *addr)

Method Description

At present, the library only guess the network address from the IP class of the machine it is running on. Of course this won't work for sub-divided network classes. And it's incredibly not simple to get the broadcast address and keep the code portable.
Use this to set the correct network broadcast address.

addr is the network broadcast address of the machine, either in network byte-order (big endian) for the integer argument, or in standard dot notation for the char* argument.


Returns 0 on success, -1 on error.
