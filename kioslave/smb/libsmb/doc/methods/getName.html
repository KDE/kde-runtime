char *getName(int fd)

Method Description

This method returns the 'basename' of the URL the file descriptor parameter refers to. That's to say only the file name will be returned, and the URL stripped.

fd should be a valid opened file descriptor.

A buffer containing the name allocated with new is returned, or 0 on error.
