char *buildURL(const char* workgroup=0, const char* host=0, const char* share=0, const char* file=0, const char* user=0)

Method Description

This will build an URL from its arguments.

workgroup should contain the workgroup part of the URL, or 0.

host should contain the host part of the URL, or 0.

share should contain the share on the host, or 0.

dir should contain the directory/file part of the URL, or 0.

user should contain the optional user name if specified.

The resulting URL is allocated with new.
