int parse(const char *name, char* &workgroup, char* &host, char* &share, char* &dir, char* &user)

Method Description

Use this method to parse an URL. The advantage of using this instead of your own routine is that :
    . You don't have to reinvent the wheel
    . This function uses internal library variables so as to determine whether 'smb://foo' represents a host or a workgroup.

All the following notations are accepted :
    . [smb:[user@][workgroup]][//host[/share[/dir[/file]]]] (deprecated)
    . [smb:][//[user@][workgroup/][host[/share[/dir[/file]]]]] (rfc 1738-compliant)
    . Same as above with \ instead of /

name should contain the URL to parse in ASCIIZ.

All the following parameters will point to a char* buffer allocated with new on return, or 0.

workgroup will contain the workgroup part of the URL

host will contain the host part of the URL

share will contain the share on the host.

dir will contain the directory/file part of the URL. No distinction is made between directories and file at this level. Use stat on the URL of you want to do this.

user will contain the optional user name if specified.

This function will return -1 on error, 0 on success.
