int cd(int dd, const char *subdir)

Method Description

Change directory function. It takes a directory descriptor and a given subdirectory name, open it, and return a directory descriptor for it.
Note that this doesn't close the initial directory descriptor.

dd should be a valid opened directory descriptor.

subdir is the name of the subdirectory to open. '.' and '..' are interpreted correctly.

On error, -1 will be returned. Otherwise an opened directory descriptor for the subdirectory will be returned.
