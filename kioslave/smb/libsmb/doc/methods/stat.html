int stat(const char *filename, struct stat *buf)

Method Description

This function is used to get information about a file/directory/URL

filename is an URL, or a name relative to the curent share.

buf points to a stat structure allocated by the user, which will contain information about the URL.

All the standard macros (ISDIR, ...) work fine on the resulting buffer, this functions performs all the conversions between DOS attributes and Unix permissions.
On error, -1 is returned and SMBIO::errno is set accordingly, otherwise 0 is returned.
