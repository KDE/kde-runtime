void setPasswordCallback(char* (*getFunc)(const char*) = 0)

Method Description

This function is used to specify a callback which will be used when the library needs to ask the user a name or password.
Note that the callback should only be called when the server requires an authentication and not for anonymous access.

GetFunc is used by the library for retrieving the user name and passwords.
It should basically print the message it is given as an argument, and return a char* answer allocated with new.
This answer will be deallocated by the library when it's not needed anymore (just being paranoïde about passwords...)
