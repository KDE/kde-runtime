class SMBdirent

Class Description

class SMBdirent : public stat
{
public:
// name of the directory entry
    char *d_name;
// Constructor and destructor
    SMBdirent();
    ~SMBdirent();
};

This class inherits the stat structure, so it contains all the st_xxxx fields.
It has been created because the SMB servers can return this information in addition to d_name, so we don't waste them.
